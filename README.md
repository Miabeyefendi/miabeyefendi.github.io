# miabeyefendi.github.io

Personal portfolio website — Letterboxd-themed, cinematic design powered by GitHub Pages.

🎬 **Live:** [miabeyefendi.github.io](https://miabeyefendi.github.io)

## About

A single-page portfolio featuring projects, social media links, and digital identity showcase with a dark cinematic Letterboxd-inspired theme.

### Features
- **Letterboxd-inspired dark theme** — #14181C background, #00E054 accent green
- **4 project showcases** — Munliker, LLM Prompts Collection, InstaDM-Scraper, Portfolio
- **30+ social platform links** — GitHub, YouTube, Twitch, Kick, Instagram, Spotify, Letterboxd, and many more
- **Glassmorphism cards** with hover animations
- **Film strip decorative elements** for cinematic aesthetics
- **Full SEO** — Open Graph, Twitter Card, Schema.org JSON-LD, sitemap, robots.txt
- **Vanilla HTML/CSS/JS** — no frameworks, no build steps, pure performance
- **Fully responsive** — mobile-first design

## File Structure

```
/
├── index.html    — Single-page application
├── style.css     — All styles with CSS custom properties
├── script.js     — Smooth scroll, Intersection Observer, dynamic year
├── robots.txt    — SEO crawler directives
└── sitemap.xml   — Google sitemap
```
